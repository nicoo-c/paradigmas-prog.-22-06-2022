Instrucciones de ejecución:

- El programa acepta tantas líneas de código como se requiera, para finalizar el ingreso se debe tipear la palabra "exit" sin las comillas.

- El programa acepta el formato "datetime NombreCuenca1 NombreCuenca2 NombreCuenca3 ... NombreCuencaN", siempre iniciando con la fecha y luego los nombres de cada cuenca, separados por un solo espacio. Ejemplo: datetime OGJS MEES ...

- Luego de esta línea, se acepta la información siguiendo el formato "Año-Mes-Día/hora:minutos InfoCuenca1 InfoCuenca2 ... InfoCuencaN", los días, meses y horas con un número de unidad, deben anteponer un 0. Ejemplo: 2023-06-22/05:06 12 33 ...
